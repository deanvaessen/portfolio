Swarm-document-agent


A markdown reader and editor for the Ethereum project.

The reader converts markdown files from the swarm network to HTML and displays it, while the editor provides for modification of the post in WYSIWYG fashion and deliver it back to the network for storage.

Word game


An experiment for a word conjugation game.

It served as a proof of concept for a larger language app, featuring a React front-end built on Websockets and Node.js, allowing for multiple simultaneous sessions that are broken up into rooms. The server also provides/manages game selection, declares winners, and so forth.

To play a match, open two browser windows and connect both to the page.
